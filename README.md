# f1tenth-gym

To get it working in Windows, you have to install WSL by wsl --install Ubuntu-24.04

Then Start WSL from the Windows search bar by searching for "Ubuntu"
Inside of the Ubuntu command window pull this repo with git clone --recurse-submodules -j8 https://github.com/ADTeam-dev/f1tenth-gym.git
Then start vscode with code .
After that, modify the devcontainer.json inside the Folder .devcontainer according to your username and filepath where you pulled the git repo to
